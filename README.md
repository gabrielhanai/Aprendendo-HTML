# Meu Primeiro Projeto em HTML

Este é o meu primeiro projeto utilizando HTML, onde desenvolvi um quiz interativo para programadores. Este projeto foi criado com o objetivo de aprender e praticar os conceitos básicos de HTML e estruturar uma página web funcional e atrativa.

## Funcionalidades

- **Estruturação de páginas web**: Utilização de elementos como `<header>`, `<main>`, `<section>`, `<footer>` para organizar o conteúdo da página.
- **Formulários interativos**: Criação de perguntas com diferentes tipos de inputs, como:
  - Múltipla escolha (`radio buttons`)
  - Resposta em texto (`text`)
  - Senha (`password`)
  - Data (`date`)
  - Seleção múltipla (`checkbox`)
  - Upload de arquivos (`file`)
  - Menu suspenso (`select`)
- **Uso de imagens**: Inclusão de imagens com a tag `<img>` e descrição com `<figcaption>`.
- **Tabelas**: Criação de uma tabela para exibir a pontuação e o feedback correspondente.
- **Lista de respostas**: Apresentação das respostas corretas em uma lista ordenada.
- **Elementos semânticos**: Uso de tags semânticas para melhorar a acessibilidade e a organização do código.

## Objetivo

Este projeto foi desenvolvido para testar e aprimorar meus conhecimentos em HTML, criando uma aplicação simples e interativa. Ele também serve como um marco inicial na minha jornada de aprendizado em desenvolvimento web.

## Como visualizar

1. Faça o download do arquivo `index.html`.
2. Abra o arquivo em qualquer navegador de sua preferência.
